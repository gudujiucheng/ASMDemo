# ASMDemo
ASM简单用法范例，主要用于学习和简单测试使用。

## module 说明
* app:主module

### 插桩简单测试module
* plugin:用于插桩简单测试的lib
* 文章介绍：https://www.jianshu.com/p/0a56e151e00b

### 异常捕获插桩module
* try_catch_plugin:测试指定方法异常捕获的lib

### 插桩实现线程监控
* thread_lib:插桩代码库
* thread_plugin：插桩插件

### 插桩实现敏感方法调用监控
* method_call_record_lib ：插桩代码库
* method_call_record_plugin：插桩插件
* 介绍：https://www.jianshu.com/p/70e14250cccc
